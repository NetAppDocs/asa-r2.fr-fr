== Configuration SAN

[cols="2,4"]
|===
| Mise à jour | Description 


| link:../learn-more/additional-supported-ontap-features.html#san-direct-attachment["Prise en charge des configurations FC à connexion directe"] | Un commutateur FC n'est pas nécessaire pour établir des connexions entre les hôtes FC et FC-NVMe et le stockage ONTAP. Les hôtes peuvent être connectés directement aux ports adaptateurs FC sur les systèmes ASA r2. Cette capacité de connexion directe s'avère particulièrement précieuse sur les sites distants et dans les environnements hybrides où la simplicité est primordiale. 
|===


== Sécurité

[cols="2,4"]
|===
| Mise à jour | Description 


| link:../secure-data/enable-anti-ransomware-protection.html["Prise en charge d'ARP/AI avec les relations de synchronisation active SnapMirror"] | La protection autonome contre les ransomwares avec intelligence artificielle (ARP/AI) prend en charge les unités de stockage principales en lecture-écriture dans les relations de synchronisation active SnapMirror sur les systèmes ASA r2. ARP/AI s'exécute sur les unités de stockage principales actives, crée des instantanés ARP uniquement primaires et fournit une détection des ransomwares basée sur l'apprentissage automatique pour ces unités de stockage. 


| link:../learn-more/additional-supported-ontap-features.html#tls-hardware-offload["Prise en charge du déchargement matériel TLS"] | Vous pouvez décharger le chiffrement et le déchiffrement Transport Layer Security (TLS) de l'ensemble de votre cluster sur des cartes Ethernet compatibles. Cela réduit la charge du processeur et améliore les performances des systèmes utilisant TLS. 
|===


== Efficacité du stockage

[cols="2,4"]
|===
| Mise à jour | Description 


| link:../learn-more/additional-supported-ontap-features.html#global-deduplication["Prise en charge de la déduplication globale"] | La déduplication globale étend les économies d'espace de déduplication ONTAP existantes à l'ensemble de la Storage Availability Zone (SAZ). En éliminant les blocs de données dupliqués sur toutes les unités de stockage de la SAZ, elle peut améliorer considérablement l'efficacité du stockage et réduire le coût de possession. La déduplication globale est activée par défaut et fonctionne de manière transparente, sans nécessiter de configuration administrative. 
|===
